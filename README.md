## ðŸ”­ Overview
This project serves as a digital storefront for a fictional coffee brand. It includes a fully navigational interface allowing users to explore coffee blends, view bakery items, locate outlets, and access a member login area. 

The primary goal of this project was to master **CSS Flexbox** for layout management and media queries for mobile responsiveness, ensuring a seamless experience across desktops, tablets, and mobile devices.

---

## âœ¨ Key Features

* **Responsive Navigation Bar:** * Features a flexible layout that adapts from a horizontal menu on desktop to a stacked, touch-friendly layout on mobile.
* **CSS Grid & Flexbox Layouts:** * utilized for the product cards (Hot/Cold coffee sections) and the footer to ensure alignment and spacing consistency.
* **Multi-Page Architecture:** * Includes distinct pages for Home (`index.html`), Bakery (`bakery.html`), About Us (`about.html`), Outlets (`outlets.html`), and Login (`login.html`).
* **Interactive Elements:** * Hover effects on buttons, navigation links, and product cards to enhance user engagement.
* **Semantic HTML5:** * Proper use of `<header>`, `<main>`, `<footer>`, and `<section>` tags for better accessibility and SEO structure.

## â˜ï¸ Deployment Strategy
This project is hosted using **GitHub Pages**, which serves static HTML/CSS files directly from the repository.

**Steps used to deploy:**
1.  Pushed all source code (`index.html`, `style.css`, etc.) to the `main` branch.
2.  Configured repository **Settings > Pages**.
3.  Selected **Deploy from branch** as the source.
4.  Set the directory to `/(root)`.
5.  GitHub Actions automatically built and deployed the site to the public URL.
